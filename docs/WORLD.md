# Ferrite Wastes

Provisional world identity for the C port of The Hollow Grid. Rename is allowed
before first public deploy if a better name sticks; keep the protocol and
federation anchors stable.

## One-line

A Grid node where magnetic memory still holds a charge the network never
collected: oxide, coil, and residual signal.

## What it is not

Not a re-skin of Hollow, Dustfall, Rust Choir, or Verdigris Spool. Same engine
contract (`docs/protocol.md` upstream), different voice, graft rooms, and
endgame stronghold.

## Shared federation requirements

Reuse upstream **canonical anchor room ids** required for travel / rescue /
cross-world smoke (see `the-hollow-grid/docs/worlds.md`). Local color is a graft
on that skeleton (working name: the **Coil Yard**), not a fork of the map graph.

## Voice notes (draft)

- Industrial / magnetic / residual-signal imagery (ferrite, hysteresis, bit rot
  as weather, not as jokes)
- Prose stays punk-direct; no corporate sycophancy
- `@event` carries truth; English is flavor
